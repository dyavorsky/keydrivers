# Key Drivers Analysis

### About

This package provides functions to facilitate execution of several methods for key drivers analysis, accommodating continuous and ordinal data. Variables are typed based on their R class: numeric vectors are treated as continuous and ordered factors as ordinal. Nominal (unordered factor) variables are not supported for the outcome or predictors. Binary variables should be coded as `0/1` numeric. Methods include:

 - list
 - as
 - developed
 

### Access

Users with access to this GitHub repo can install the package with

```         
remotes::install_github("dyavorsky/keydrivers")
```
